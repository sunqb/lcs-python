# 课程7：列表容器 - 存储多个东西

## 学习目标
- 学习什么是列表
- 了解如何操作列表
- 制作购物清单程序

## 什么是列表？
列表就像一个大盒子，可以装很多东西。

```python
水果 = ["苹果", "香蕉", "橙子"]
数字 = [1, 2, 3, 4, 5]
```

## 基本操作
```python
# 获取列表元素
我的水果 = ["苹果", "香蕉", "橙子"]
print(我的水果[0])  # 第一个：苹果

# 添加元素
我的水果.append("葡萄")
print(我的水果)  # ['苹果', '香蕉', '橙子', '葡萄']
```

## 练习项目：购物清单
```python
购物清单 = []
购物清单.append("牛奶")
购物清单.append("面包")
购物清单.append("鸡蛋")

print("=== 我的购物清单 ===")
for i, 商品 in enumerate(购物清单, 1):
    print(f"{i}. {商品}")
```
