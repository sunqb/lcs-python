## 数字在Python中

在Python中，你可以直接使用数字来做数学计算，就像你在学校学的数学一样。

## 基本算术运算

让我们看看如何使用Python进行基本的数学运算。

### 加法

加法使用`+`符号：

```python
print(3 + 2)  # 输出：5
```

### 减法

减法使用`-`符号：

```python
print(5 - 1)  # 输出：4
```

### 乘法

乘法使用`*`符号：

```python
print(4 * 2)  # 输出：8
```

### 除法

除法使用`/`符号：

```python
print(6 / 2)  # 输出：3.0
```

## 综合示例

让我们写一个程序，计算你买3个苹果，每个苹果2元，总共需要多少钱。

```python
苹果数量 = 3
每个苹果价格 = 2
总价 = 苹果数量 * 每个苹果价格
print("总共需要", 总价, "元")
# 输出：总共需要 6 元
```

### 如何运行这个程序？

1. 打开文本编辑器，复制上面的代码。
2. 将文件保存为`apple_cost.py`。
3. 在终端中输入`python apple_cost.py`并按回车键。
4. 你会看到计算结果。
